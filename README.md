# Banco de Dados para Supermercado

Projeto acadêmico de banco de dados relacional para gerenciamento de um supermercado, desenvolvido utilizando MySQL.

## Objetivo

Desenvolver uma solução para armazenar e gerenciar informações relacionadas a clientes, produtos, vendas, estoque e colaboradores, aplicando conceitos de modelagem de dados e banco de dados relacionais.

## Atividades Realizadas

* Levantamento de requisitos e análise das regras de negócio.
* Modelagem conceitual do banco de dados.
* Modelagem lógica do banco de dados.
* Implementação da estrutura em MySQL.
* Documentação e apresentação do projeto.

## Arquivos do Projeto

* Levantamento de Requisitos
* Modelo Conceitual
* Modelo Lógico
* Script SQL do Banco de Dados

## Tecnologias Utilizadas

* MySQL
* SQL
* brModelo
